# Controle-Financeiro
# 💰 Controle Financeiro Inteligente

Aplicação desenvolvida em Python com Streamlit para organização financeira pessoal e empresarial.

## 🚀 Funcionalidades

- 📂 Upload de faturas de cartão (PDF)
- 🧠 Classificação automática de gastos
- ✏️ Edição manual de categorias
- 💰 Lançamentos manuais (receitas e despesas)
- 📊 Relatórios financeiros em PDF
- 📋 Histórico completo de movimentações
- 🗑 Exclusão de lançamentos
- 🔎 Filtro por mês e ano

## 🎯 Objetivo

Facilitar o controle financeiro de forma simples, visual e automatizada.

## 🛠 Tecnologias utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- ReportLab
